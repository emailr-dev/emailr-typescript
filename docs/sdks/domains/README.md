# Domains

## Overview

Domain verification and management

### Available Operations

* [add](#add) - Add domain
* [list](#list) - List domains
* [verify](#verify) - Verify domain
* [getDnsStatus](#getdnsstatus) - Get DNS status
* [update](#update) - Update domain
* [delete](#delete) - Delete domain

## add

Add a new domain with BYODKIM

### Example Usage

<!-- UsageSnippet language="typescript" operationID="addDomain" method="post" path="/v1/domains" -->
```typescript
import { Emailr } from "emailr";

const emailr = new Emailr({
  bearerAuth: process.env["EMAILR_BEARER_AUTH"] ?? "",
});

async function run() {
  const result = await emailr.domains.add({
    domain: "example.com",
  });

  console.log(result);
}

run();
```
