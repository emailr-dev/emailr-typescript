### Available Operations

* [create](#create) - Create segment
* [list](#list) - List segments

## create

Create a new contact segment

### Example Usage

<!-- UsageSnippet language="typescript" operationID="createSegment" method="post" path="/v1/segments" -->
```typescript
import { Emailr } from "emailr";

const emailr = new Emailr({
  bearerAuth: process.env["EMAILR_BEARER_AUTH"] ?? "",
});

async function run() {
  const result = await emailr.segments.create({
    name: "Active Users",
    description: "Users who opened an email in the last 30 days",
    conditions: {
      "opened_email": {
        "within_days": 30,
      },
    },
  });

  console.log(result);
}

run();
```
